# intel DG1

最近开始研究视频的编解码,所以搞来了一个显卡

这里之所以选择 大哥1 主要是因为以下几个优点:

- 功耗够低,无需外接供电(待机功耗低至于数W,满载也仅有40W)
- 战力足,性能好
- 性价比高(我入手的时候大约200元左右)
> 这个是最重要的一点,200元这个级别能选择的卡太少了
